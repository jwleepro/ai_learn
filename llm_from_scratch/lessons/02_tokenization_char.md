# 02. 글자 토큰화(텍스트를 숫자로 바꾸기)

목표: 텍스트를 모델이 다룰 수 있는 **정수 ID 시퀀스**로 바꾸고, 다시 텍스트로 되돌리는 과정을 이해합니다.

---

## 1) 왜 숫자로 바꾸나?

모델은 결국 숫자(벡터/행렬)로 계산합니다. 그래서:

- 텍스트 → 토큰(글자) → **정수 ID** → (임베딩으로) **벡터**

이 흐름이 기본입니다.

---

## 2) 가장 단순한 방식: “등장한 글자 목록” 만들기

예를 들어 텍스트가:

```
hello
```

등장 글자 집합(vocab)은 `{h, e, l, o}` 같은 느낌이고,

- `h -> 0`
- `e -> 1`
- `l -> 2`
- `o -> 3`

처럼 번호를 붙입니다.

> 주의: 한국어/이모지처럼 유니코드 글자가 섞여도 “글자 기반”은 돌아갑니다.  
> 다만 vocab이 커질 수 있어서, 나중에 BPE 같은 서브워드 토크나이저로 넘어갑니다.

---

## 3) 필요한 2가지 함수

토크나이저는 최소 다음 두 함수를 제공하면 됩니다.

- `encode(text) -> list[int]`
- `decode(ids) -> str`

이 두 개만 있으면 학습과 생성 모두 진행 가능합니다.
